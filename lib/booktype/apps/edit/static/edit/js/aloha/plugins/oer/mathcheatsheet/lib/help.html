Help using the math editor
To add math to your document, type math in the text field using either ASCIIMath or LaTeX notation. The display math will be shown below in real time. When you're done, click "Done" or click anywhere outside the blue box.
  • ASCIIMath is a simple input notation similar to that of a graphing calculator. For example, x^(ab) would display as xab. Learn more.
  • LaTeX, while similar to ASCIIMath, includes some more complex notation for advanced math. Learn more.
